Project Description

This app is a Pokemon trading card index-fund prototype. It calculates a live index price by combining card prices proportionally to each card’s basket contribution, and then lets you explore history, basket composition, and paper-trade the index funds.

How Pricing Works

Index Price = Σ(card price × basket contribution % / 100)

Prices are built from card-level market data and persisted in cache/history so graphs and index views still work when live calls are temporarily unavailable.

What This Solves

Most card investing is concentrated and operationally heavy. You need to choose individual cards, manage storage, and accept single-card risk if one position underperforms. This project shifts that model to index-style exposure so users track a weighted basket instead of one collectible.

The broad market index provides diversified exposure, while era-focused index funds let users target specific periods such as Diamond and Pearl, Platinum, and HeartGold and SoulSilver.

How To Use The App

1. Open Index Marketplace to compare index funds. 2. Select a fund to inspect history and constituents. 3. Use Basket Builder to see set-level contribution and weighted value. 4. Use Paper Trading to simulate buy and sell actions. 5. Use Data Cache to validate cache and database status.

Project Overview

This app is a Pokemon card index-fund prototype. It tracks market movement by aggregating card prices into weighted baskets and exposing each basket as a single index price.

How The Index Works

Index Price = sum(card price * basket contribution % / 100)

Each card contributes according to its basket weight. This gives diversified exposure across multiple cards and sets, rather than relying on one collectible.

What You Can Do

1. Compare index funds in Index Marketplace. 2. Inspect card-level composition in Fund Explorer. 3. Review set-level weighting in Basket Builder. 4. Analyze trends in Price History. 5. Simulate trades in Paper Trading and monitor results in Portfolio.
